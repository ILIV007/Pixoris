📚 راهنمای رفع مشکل

اگر GitHub 403 میده:

کنترل کن که GITHUB_TOKEN secrets ست شده باشه و دسترسی repo:write داشته باشه. v3.1 این مشکل رو با اضافه کردن User-Agent header حل کرده.

اگر Schema ناسینک بود:

دستور wrangler d1 execute pixoris-db --file=./schema.sql --remote رو اجرا کن.

اگر JWT_SECRET ست نبود:

دستور wrangler secret put JWT_SECRET رو اجرا کن (حداقل ۳۲ کاراکتر).
